Datos Técnicos:
Si sos interesado en los datos, te cuento que esta aplicació se desarrolló usando los siguientes recursos y herramientas:
  • Lenguaje de programación Python 2.6
  • Biblioteca gráfica wxPython
  • Shell interactiva ipython
  • Biblioteca de reportes Reportlab
  • Biblioteca de conexión MySQLdb
  • Base de datos MySQL
  • Administrador gráfico PHPMyAdmin
  • Servidor web Apache
  • Trac cedido gentilmente por USLA(Usuarios de Software Libre Argentina)
  • Editor Geany
  • Aplicación Sunbird de agenda y calendarios
  • Editor Vim
  • Editor HTML Bluefish
  • Editor de interfaces gráficas wx Glade
  • Todo corriendo en Ubuntu GNU/Linux y eeebuntu GNU/Linux
